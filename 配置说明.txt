旅游计划生成器 - 配置说明
====================================

1. 安装Python依赖
   在命令行中运行：
   pip install -r requirements.txt

2. 配置API密钥
   - 创建一个名为 .env 的文件（在travel-planner目录下）
   - 在文件中添加以下内容：
     DEEPSEEK_API_KEY=你的DeepSeek API密钥
     GOOGLE_API_KEY=你的Google API密钥（可选）
     GOOGLE_SEARCH_ENGINE_ID=你的Google搜索引擎ID（可选）
   
   例如：
   DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   GOOGLE_API_KEY=AIzaSyxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   GOOGLE_SEARCH_ENGINE_ID=xxxxxxxxxxxxxxxxxxxxxxxxx

3. 获取DeepSeek API密钥
   - 访问 https://platform.deepseek.com/
   - 注册/登录账号
   - 进入 API Keys 页面
   - 创建新的API密钥
   - 复制密钥到.env文件中

4. 获取Google Custom Search API密钥（可选，用于搜索功能）
   - 访问 https://console.cloud.google.com/
   - 创建新项目或选择现有项目
   - 启用 "Custom Search API"
   - 创建 API 密钥
   - 访问 https://programmablesearchengine.google.com/ 创建搜索引擎
   - 获取搜索引擎 ID (CX)
   - 将 API 密钥和搜索引擎 ID 添加到 .env 文件中
   
   注意：如果不配置Google API密钥，应用仍可正常使用，但不会在生成计划时自动搜索最新信息。Google Custom Search API每天有100次免费搜索配额。

5. 运行应用
   - 方法1：双击 start.bat 文件（Windows）
   - 方法2：在命令行中运行 python app.py

6. 访问应用
   - 打开浏览器
   - 访问 http://localhost:5000

注意事项：
- 使用DeepSeek API会产生费用，请查看DeepSeek的定价页面
- DeepSeek API价格实惠，性价比高
- Google Custom Search API 每天有100次免费搜索配额
- 确保网络连接正常
- 如果遇到错误，请检查.env文件是否正确配置
- 谷歌搜索功能是可选的，不配置也能正常使用应用
